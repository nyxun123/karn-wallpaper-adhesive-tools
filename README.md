# KARN Wallpaper Adhesive Technical Tools

A free, static mixing-ratio calculator and printable sample-test record for wallpaper adhesive powder.

The tool:

- calculates water mass from a user-supplied product ratio;
- records matched test conditions for batch or supplier comparisons;
- does not collect, transmit, or store user data;
- links to product-specific KARN resources for bulk supply, technical documents, and OEM workflows.

Public tool: <https://nyxun123.github.io/karn-wallpaper-adhesive-tools/>

Official manufacturer website: <https://kn-wallpaperglue.com/en/>

## Local preview

Serve the directory with any static HTTP server, for example:

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173/>.

## Scope

This tool does not recommend a universal mixing ratio. Users must enter the product-specific ratio stated on the current label or technical data sheet. It does not replace local regulations, workplace safety procedures, or onsite testing.
